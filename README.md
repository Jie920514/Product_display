# 🛒 商品展示系統

這是一個基於 .NET 8 開發的現代化商品管理與展示平台，強調流暢的用戶體驗與高效的資料處理。

## 🛠 使用框架及技術
- **後端**: `C#` / `ASP.NET Core MVC` / `.NET 8`
- **資料庫**: `SQL Server` / `Entity Framework Core (ORM)`
- **前端**: `Razor Pages` / `Bootstrap 5` / `JavaScript (Fetch API)`
- **狀態管理**: `Session-based` 登入驗證

## 🚀 核心功能

### 1. 商品展示與搜尋
* **Notion 風格佈局**: 採用自定義卡片排版，提供簡潔直觀的視覺體驗。
* **動態過濾**: 實作關鍵字搜尋功能，快速篩選目標商品。
* **穩定性**: 具備圖片失效處理機制 (`onerror fallback`)，確保頁面美觀。

### 2. 非同步收藏系統 (Ajax/Fetch)
* **即時互動**: 採用 **Fetch API** 呼叫後端接口，達成不跳轉網頁（No-refresh）即可切換收藏狀態。
* **智能渲染**: 首頁載入時自動比對資料庫，動態呈現愛心狀態（實心/空心）。
* **UX 動畫**: 在「我的最愛」頁面移除商品時，加入 **CSS 漸隱與縮放動畫**。

### 3. 資料庫關聯設計
* **效能優化**: 使用 **LINQ Join** 語法，單次請求即可取得使用者收藏之完整商品資訊。
* **架構設計**: 實作多對多關係的中間表 (`UserFavorites`)，確保資料結構嚴謹。
